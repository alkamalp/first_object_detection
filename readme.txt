THIS IS A PROJECT ABOUT REAL TIME WITH WEBCAM OBJECT DETECTION USING OPENCV

===HOW TO USE THIS PROJECT===
1. Download this project as zip
2. Next unzip the file
3. Open the first_project.py, if you want to run the basic object detection file
4. Or you could open the second_project, if you want to run the remove duplication version
5. Then you could run the file as usual and enjoy!!!
